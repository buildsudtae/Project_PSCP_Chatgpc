# 🎲 A Board (Chatgpc) — เว็บสำหรับยืมบอร์ดเกม

เว็บแอปพลิเคชันสำหรับยืมบอร์ดเกมของคณะเทคโนโลยีสารสนเทศ KMITL แสดงสถานะการยืมแบบ **Real-time** พร้อมลิงก์ยืมบอร์ดเกมและวิธีการเล่น

> 🎥 คลิปสาธิต: [https://youtu.be/Q9SQLhhDYGw](https://youtu.be/Q9SQLhhDYGw?si=ulgPyOhn1IWuJOVG)

---

## 🎯 ที่มาและวัตถุประสงค์

คณะ IT KMITL มีตู้บอร์ดเกมให้นักศึกษายืมใช้งาน แต่ระบบเดิมมีปัญหา **สถานะการยืมไม่อัปเดตแบบ real-time** ทำให้เกิดความสับสน โปรเจกต์นี้จึงพัฒนาเว็บใหม่เพื่อ:

- แสดงสถานะการยืมบอร์ดเกมแบบ real-time
- เชื่อมต่อกับลิงก์ยืมบอร์ดเกมของคณะ (Google Form)
- ให้ผู้ใช้ตรวจสอบได้ทันทีว่าบอร์ดเกมที่ต้องการมีคนยืมอยู่หรือไม่

## ✨ ฟีเจอร์หลัก

| ฟีเจอร์ | คำอธิบาย |
|---------|----------|
| 🟢 แสดงสถานะบอร์ดเกม | กรอกเวลา (นาที) → กด "ยืมบอร์ดเกม" → รูปจางลงพร้อมเครื่องหมาย ✓ และนับถอยหลัง |
| 📋 ลิงก์ยืมบอร์ดเกม | กด "ยืมบอร์ดเกม" เพื่อเปิด Google Form ในแท็บใหม่ |

| 🔍 ค้นหาบอร์ดเกม | พิมพ์ชื่อบอร์ดเกม → กด Search → แสดงเฉพาะบอร์ดเกมที่ค้นหา |

## 🛠️ เทคโนโลยีที่ใช้

**Front-end**
- **HTML** — โครงสร้างหน้าเว็บ
- **CSS** — ตกแต่งและจัดวาง
- **JavaScript** — ควบคุมการทำงาน

**Back-end**
- **Python + Flask** — Web framework จัดการ routing และ render template

## 🚀 วิธีติดตั้งและรันโปรเจกต์

### สิ่งที่ต้องมี
- Python 3.x
- Flask (`pip install flask`)

### ขั้นตอน

```bash
# 1. Clone repository
git clone https://github.com/Buildarwitsakorn/PSCP_Chatgpc.git
cd PSCP_Chatgpc

# 2. ติดตั้ง dependencies
pip install flask

# 3. รันโปรแกรม
python app.py
```

4. เปิด browser แล้วไปที่ **http://localhost:5500**

## 📖 วิธีใช้งาน

1. **ดูสถานะบอร์ดเกม** — สังเกตรูปภาพแต่ละเกมว่ายังว่างอยู่หรือถูกยืมไปแล้ว
2. **ยืมบอร์ดเกม** — กรอกเวลาที่ต้องการใช้ (นาที) แล้วกดปุ่ม **"ยืมบอร์ดเกม"**
3. **ดูวิธีเล่น** — กดปุ่ม **"วิธีการเล่น"** เพื่อเปิดคลิป YouTube
4. **ค้นหาเกม** — พิมพ์ชื่อบอร์ดเกมในช่อง Search แล้วกด **"Search"**
5. **About** — คลิกที่เมนู About เพื่อดูคำแนะนำการใช้งาน
